Star Citizen Kill Tracker LITE

Introduction

SC Kill Tracker LITE is a lightweight application designed to track player vs player (PvP) and player vs environment (PvE) kills in Star Citizen by monitoring the game's log files. The application provides real-time kill tracking, statistics, and an optional overlay display.

Key Capabilities

  • Real-time Kill Tracking: Monitors Star Citizen's Game.log file for kill events
  • Kill Classification: Automatically distinguishes between PvP, PvE, vehicle kills, and other death types
  • NPC Recognition: Identifies NPCs vs real players
  • Overlay Display: Optional on-screen overlay showing recent kills
  • Statistics & History: Tracks and stores kill data locally
  • Multiple Channels: Supports LIVE, PTU, HOTFIX, and EVOCOTI game channels
  • License Key System: Secure registration and validation system

Understanding Windows SmartScreen Warning

What is SmartScreen?

When you first run SCKillTrac-LITE.exe, Windows Defender SmartScreen may display this warning:

Microsoft Defender SmartScreen prevented an unrecognized app from starting.
Running this app might put your PC at risk.

App: SCKillTrac-LITE.exe
Publisher: Unknown publisher

Why Does This Happen?

  1. Unsigned Executable: The application is not digitally signed with a code signing certificate.
  2. Unknown Publisher: Microsoft hasn't verified the publisher through their reputation system.
  3. First Launch: Windows shows this for any newly downloaded executable from an unknown source.

Is It Safe?

✅ Yes, it is safe to run. This is a standard Windows security feature that appears for unsigned applications. The warning is expected for applications not distributed through the Microsoft Store or signed with a commercial certificate.

How to Proceed

  1. Click "More info" to see additional details.
  2. Click "Run anyway" to launch the application.
  3. After the first successful launch, Windows will remember your choice for future runs.

Adding Antivirus Exclusions

Why Add Exclusions?

Some antivirus software may flag SCKillTrac-LITE.exe as potentially unwanted software or block it from running. This happens because the application monitors file changes and interacts with the game's log files. Adding an exclusion prevents false positives and ensures smooth operation.

Windows Defender

  1. Open Windows Security (search for "Windows Security" in Start menu)
  2. Go to Virus & threat protection
  3. Click Manage settings under Virus & threat protection settings
  4. Scroll down to Exclusions and click Add or remove exclusions
  5. Click Add an exclusion → File
  6. Navigate to and select SCKillTrac-LITE.exe
  7. Alternatively, add the entire folder where the application is located

Other Antivirus Software

For other antivirus programs (Norton, McAfee, Avast, AVG, etc.):

  1. Open your antivirus software's settings or control panel
  2. Look for Exclusions, Exceptions, or Whitelist settings
  3. Add SCKillTrac-LITE.exe to the exclusions list
  4. You may also need to exclude the application's data folder if it's located separately

What to Exclude

  • Executable File: SCKillTrac-LITE.exe
  • Application Folder: The folder containing the executable (optional but recommended)
  • Data Folder: If stored separately, exclude the application's data/config folder

Still Getting Blocked?

If your antivirus continues to block the application after adding exclusions:

  • Ensure you've added both file and folder exclusions
  • Restart your antivirus software after adding exclusions
  • Check if your antivirus has separate settings for "Real-time protection" and "On-demand scanning"
  • Some antivirus programs require administrator privileges to modify exclusions
  • Contact support at discord.sckilltracker.com if issues persist

Installation & First Launch

System Requirements

  • Operating System: Windows 10/11 (64-bit)
  • Star Citizen: Must be installed and have generated Game.log files
  • Internet Connection: Required for license validation

Installation Steps

  1. Download the SCKillTrac-LITE.exe file.
  2. Place the file in your desired location (e.g., Desktop).
  3. Run the executable (handle the SmartScreen warning as described above).

First Launch Setup

1. Account Registration

You'll need to create an account with a Username and a License Key in the format SCKT-LTE-XXX-XXXXX. You can get a key from discord.sckilltracker.com.

2. Game Log Detection

The application automatically searches for Star Citizen's Game.log file. If not found, you can manually specify the path in Settings.

Main Features

  • Kill Tracking: Monitors the Game.log for player kills, NPC kills, vehicle kills, and deaths.
  • Kill Classification: Automatically categorizes kills into PvP, PvE, Vehicle, and Other.
  • Overlay Display: An optional, customizable on-screen overlay to show recent kills and stats.
  • Statistics & History: Tracks total kills, K/D ratio, and session statistics.
  • Game Log Viewer: A built-in tool to inspect the Game.log file directly.
  • System Messages: Receive notifications about updates, maintenance, and announcements.

Settings & Configuration

Accessing Settings

Open Settings from the main menu, the system tray icon, or a configured keyboard shortcut.

1. Overlay Settings

Customize the overlay's position, size, and display options. You can also lock its position to prevent accidental movement.

2. Log File Settings

Manually set the path to your Game.log file and switch between game channels like LIVE, PTU, etc.

3. Application Settings

Configure startup options (e.g., launch with Windows) and enable offline mode, which provides a 24-hour grace period for license validation.

4. Advanced Settings

Enable debug mode, clear application caches, or reset all settings to their default values.

Troubleshooting

License Key Validation Failed

Solutions: Check your internet connection, verify the key format (SCKT-LTE-XXX-XXXXX), and contact support if the issue persists. The app has a 24-hour offline grace period.

Game Log Not Found

Solutions: Ensure Star Citizen is installed and has run at least once. Manually set the path in Settings and verify the correct game channel (LIVE/PTU) is selected.

Kills Not Being Tracked

Solutions: Verify the log path is correct, the game is running, and that kill tracking is enabled in Settings. Use the debug overlay to check log monitoring status.

Antivirus Blocking or Flagging the Application

Solutions: Add SCKillTrac-LITE.exe to your antivirus exclusions list. See the Adding Antivirus Exclusions section for detailed instructions. Common antivirus software may flag the application due to file monitoring behavior, but this is a false positive.

Generate Support Information

For complex issues, generate a support file via Settings → Support → Generate Support Info. This file gathers all necessary technical details to help diagnose the problem.
